# weewx-clearskies-dashboard

Phase 1 placeholder — real content lands during Phase 3 of the Clear Skies project.

Modern React + Vite single-page weather dashboard for [weewx](https://github.com/weewx/weewx). Tailwind v4 + shadcn/ui + Recharts + Lucide + Weather Icons.

Part of [Clear Skies](https://github.com/inguy24/weewx-clearskies-stack), a modular modern weather UI for weewx.
